# Game Stats Dashboard

## Description
A simple and user-friendly dashboard to search for video games and view details such as release dates, ratings, and platforms. Built using the RAWG API.

## Features
- Search for games by name.
- Displays game details dynamically.
- Responsive and gamer-themed UI.

## API Documentation
This project uses the [RAWG API](https://rawg.io/apidocs).
